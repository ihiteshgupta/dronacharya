# Dronacharya Brand Assets

## Logo Files

- `logo.svg` - Primary logo with dark background (for dark themes)
- `logo-light.svg` - Logo without background (for light themes)
- `favicon.svg` - Simplified logo for favicon

## Brand Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Deep Indigo | `#1e1b4b` | Primary brand color, backgrounds |
| Saffron Gold | `#f59e0b` | Accent, CTAs, highlights |
| Teal | `#14b8a6` | Success states, progress |
| Indigo | `#6366f1` | Interactive elements |

## Logo Concept

The logo combines:
- **Bow and Arrow**: Represents Dronacharya's legendary skill as an archer and teacher
- **Neural Network Nodes**: Represents AI and interconnected learning

## Typography

- **Display**: Plus Jakarta Sans
- **Body**: Inter
- **Code**: JetBrains Mono

## Usage Guidelines

1. Always maintain the logo's aspect ratio
2. Minimum size: 32x32px for icon, 120px width for full logo
3. Use gold (#f59e0b) for text on dark backgrounds
4. Use indigo (#1e1b4b) for text on light backgrounds
